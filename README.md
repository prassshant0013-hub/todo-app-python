# 📝 Pro To-Do App (Python GUI)

A modern To-Do List application built using Python and Tkinter.
This project features a clean dark UI, task prioritization, and persistent storage using JSON.

## 🚀 Features

* ✔️ Add and delete tasks
* ✔️ Mark tasks as completed
* ✔️ Task priority system (Low, Medium, High)
* ✔️ Filter tasks (All / Done / Pending)
* ✔️ Keyboard shortcuts support
* ✔️ Data saved locally (JSON file)
* ✔️ Clean and responsive dark UI
